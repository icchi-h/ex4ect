#express4 template for ect

## How to Use

### install and edit
```
$ git clone git@github.com:n0bisuke/ex4ect.git
$ cd ex4ect
$ npm i
$ DEBUG=myapp ./bin/www
  myapp Express server listening on port 4444 +0ms
```

### access

http://localhost:4444
